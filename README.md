# Dr. Maya Reynolds Therapy Website

A professional therapy website for Dr. Maya Reynolds, PsyD, built with Next.js and Tailwind CSS. This project demonstrates frontend development skills by cloning and redesigning a template website with custom content, theme, and functionality.

## Features

- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **SEO Optimized**: Includes relevant keywords for anxiety therapy and trauma counseling in Santa Monica
- **Contact Form**: Functional contact form for client inquiries
- **Custom Sections**: Includes a unique "Our Office" section showcasing the therapy space

## Tech Stack

- **Framework**: Next.js 
- **Styling**: Tailwind CSS
- **Language**: JavaScript (ES6+)
- **Deployment**:Vercel


## Customization

### Color Theme

The site uses a custom color palette defined in `tailwind.config.js`:

- Primary: Slate gray (#4A5568) - Professional and calm
- Secondary: Light gray (#EDF2F7) - Clean backgrounds
- Accent: Soft green (#68D391) - Healing and growth

### Content

All content is based on Dr. Maya Reynolds' professional profile, including:
- Services: Anxiety therapy, trauma counseling, burnout support
- Location: Santa Monica, CA with telehealth options
- Approach: CBT, EMDR, mindfulness-based techniques
